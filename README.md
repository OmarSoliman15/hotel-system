# Rails Hotel Reservation System


## Getting Started

### Requirements
- Ruby 3.1,2
- Rails 7.1.2

### Install

```
git clone https://github.com/OmarSoliman15/hotel-system.git
cd hotel-system

bundle install
rails db:create
rails db:migrate
rails db:seed

rails s
```

You can find the insomnia's api docs from here:
[hotel-system.json](hotel-system.json)
